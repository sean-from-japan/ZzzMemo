# 一括インポート機能 利用ガイド

## 概要

Claude や ChatGPT などの AI が生成したタスクリストを、コピー＆ペーストで qcatch に一括登録できる機能です。

---

## 使い方

1. **Inbox タブ**を開く
2. ツールバーの **📋 一括インポート** ボタンをクリック
3. AI が生成したリストをテキストエリアに貼り付ける
4. **プレビュー** ボタンをクリック → パース結果を確認
5. 不要な項目は **✕** で削除
6. **インポート実行** でタスクを一括追加

---

## 対応フォーマット

以下の形式をすべて自動認識します。

### 箇条書き

```
- タスクA
* タスクB
• タスクC
```

### 番号付きリスト

```
1. タスクA
2. タスクB
1) タスクC
```

### GitHub チェックリスト形式

```
- [ ] 未完了タスク
- [x] 完了済み（インポート対象外・スキップ）
```

### プレーンテキスト（1行1タスク）

```
タスクA
タスクB
```

---

## カテゴリの自動検出

### セクションヘッダー（以降の行に適用）

```
## 仕事
- メール返信
- 会議資料作成

## プライベート
- 歯医者の予約
```

対応形式: `## カテゴリ名` / `**カテゴリ名**` / `【カテゴリ名】` / `カテゴリ名:`

### 行末タグ（その行のみ）

```
- メール返信（仕事）
- 牛乳を買う (買い物)
- 英語の勉強 [学習]
```

利用可能なカテゴリ: `仕事` / `プライベート` / `買い物` / `学習` / `その他`

---

## 期日の指定

行末に `期日:YYYY/MM/DD` を付けると due_date として登録されます。

```
- 企画書提出 期日:2026/04/01
- 請求書送付（仕事） 期日:2026/03/31
```

---

## 実際の使用例

### AI へのプロンプト例

> 来週の仕事タスクを以下の形式でリストアップしてください：
> - タスク名（カテゴリ） 期日:YYYY/MM/DD

### AI の出力例

```
## 仕事
- 週次レポート作成 期日:2026/03/27
- クライアントへのメール返信
- 企画書レビュー 期日:2026/03/28

## プライベート
- ジム予約
- 読書：1章分
```

このテキストをそのままコピーして一括インポートに貼り付けると、カテゴリと期日が自動設定された状態でプレビューされます。

---

## 注意事項

- カテゴリなしの場合 → `inbox`（未分類）として登録
- カテゴリあり → `todo` として登録
- `- [x]` の完了済み行はスキップ
- 空行・2文字未満の行はスキップ
- プレビュー画面で内容を確認してから実行してください

---

## API（上級者向け）

```http
POST /api/tasks/bulk
Content-Type: application/json

[
  {"text": "メール返信", "category": "仕事", "due_date": "2026-04-01T00:00:00"},
  {"text": "牛乳を買う", "category": "買い物"},
  {"text": "未分類タスク"}
]
```

レスポンス: `{"added": 3}`
